## CN2 是什么？先把这事儿搞懂

简单说，CN2 是中国电信推出的一套精品骨干网络，全称 China Telecom Next Carrier Network（CTGNet）。和普通的 163 骨干网相比，CN2 的路由节点更少、时延更低、拥塞更轻，用白话说就是：**同样的钱，晚高峰体验差距可能是天壤之别**。

但 CN2 本身还分两档：

**CN2 GT（Global Transit）**：去程和回程都会混入部分 163 骨干网节点，高峰时段偶尔会出现明显波动。价格相对便宜，是入门级选择。现在搬瓦工的 CN2 GT 套餐已经全部下架，取而代之的是 KVM 基础套餐走普通线路。

**CN2 GIA（Global Internet Access）**：全程走 CN2 精品线路，去回程都不绕弯，带宽更独立。这是搬瓦工目前主推的高端线路，分为两个子系列：DC9 CN2 GIA（洛杉矶 DC9 机房，1Gbps 带宽）和 DC6 CN2 GIA-E（洛杉矶 DC6 机房，带宽起步 2.5Gbps 甚至更高，E 代表 E-Commerce 电子商务）。

目前你在搬瓦工能买到的「搬瓦工 CN2」套餐，实际上就是 **CN2 GIA-E 系列**（E-Commerce VPS），这也是大多数人说"搬瓦工 CN2"时默认指的那个。
